# mvp
This is An Android MVP Sample. <br/>

### Usage
apply apt plugin
```
    dependencies {
        ...
        classpath 'com.neenbedankt.gradle.plugins:android-apt:1.8'
    }
```
```
    apply plugin: 'com.neenbedankt.android-apt'
```
add dependencies
```
    compile 'me.sfce:mvp:2.0.0'
    apt 'com.jakewharton:butterknife-compiler:8.2.1'
```
